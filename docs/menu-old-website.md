# Future Streamlined Menu (Target)

This is the curated, streamlined menu we are working towards. ~30 best-selling items in
6 focused categories — clean names, simple flat pricing, no item codes.

**Status:** FUTURE TARGET — switch back to this once the physical menu is updated to match.

**Why this menu is better for the website:**
- Fewer choices = faster decisions for customers
- Flat pricing = no confusion about protein tiers
- Curated selection = only the best sellers and signatures
- Clean names = no item codes cluttering the display
- 6 categories = simpler navigation
